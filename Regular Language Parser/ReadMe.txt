The files are structured in this format:
Section 1: This section contains the transition function for the automata. Each line in the section reads like this:
<current state> <space> <input symbol> <next state>
Section 2: Immediately follows, after a line of asterisks (*****), the initial state
Section 3: Immediately after another line of asterisks (*****), contains the set of accept/final states, each value represents an
accept state. The accept states in the set are also separated by blank spaces
